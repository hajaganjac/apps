Design a premium mobile networking application interface for Playgrounds Festival, a leading event for digital artists, animators, motion designers, filmmakers, and game creators.

The goal is to redesign the networking experience for creatives attending Playgrounds events, enabling meaningful professional connections based on creative disciplines, talks, and shared interests rather than random profiles.

The final interface must feel like a high-end creative platform used by professionals in the digital art industry.

The design should feel:

• cinematic
• editorial
• expressive
• bold
• premium
• artistic but highly usable

The interface must look like an official extension of the Playgrounds brand, not a generic mobile product.

⸻

Visual DNA (Critical)

The UI must follow the Playgrounds visual language.

Reference:

https://weareplaygrounds.nl/

Core design DNA:

• dark immersive interface
• cinematic neon gradients
• bold editorial typography
• layered card compositions
• experimental but controlled layouts
• strong visual hierarchy
• minimal UI chrome

The UI should feel like a creative festival publication translated into a mobile interface.

The aesthetic should resemble a hybrid of:

• a creative conference platform
• a digital art showcase
• a curated networking tool

Avoid:

• startup-style minimal UI
• generic social networking apps
• template mobile UI kits
• swipe interactions
• dating app patterns

The interface must feel designed for artists and designers.

⸻

Color System

Base interface:

Deep black background.

Primary palette:

• Neon magenta accents
• Electric purple gradients
• White typography
• Subtle glow highlights

Gradients should feel cinematic and atmospheric.

Use:

• soft glow edges
• layered gradients
• subtle light diffusion

Do NOT introduce colors outside the Playgrounds palette.

Avoid:

• bright rainbow colors
• emojis
• playful visual elements

The interface must remain professional and refined.

⸻

Typography System

Typography is a primary design element.

Use strong typographic hierarchy:

Hero titles
Large expressive headings
Medium section headings
Minimal body text

Typography should feel similar to:

• creative festival posters
• editorial magazines
• design publications

Headlines should feel impactful and expressive.

Secondary information should remain minimal and readable.

Typography should guide navigation more than icons.

⸻

Layout & Spatial System

Avoid simple vertical stacking.

The layout should follow editorial composition principles:

• asymmetric layouts
• layered card stacking
• overlapping UI blocks
• mixed card sizes
• editorial spacing rhythm

Use large whitespace zones to create visual breathing space.

Layouts should feel curated and artistic, not algorithmic.

Include:

• floating cards
• overlapping gradients
• layered sections

The interface should feel visually dynamic and immersive.

⸻

Component System (Important)

The UI must include reusable components.

Key components:

Festival Hero Card
Creative Circle Card
Attendee Profile Card
Skill Tag Component
Connect Action Button
Portfolio Link Module
Minimal Chat Bubble

Component characteristics:

Rounded corners: medium
Soft glow borders
Gradient accents
Subtle elevation shadows

Cards should feel premium and tactile.

Avoid:

• standard iOS cards
• flat UI blocks
• simple list layouts

Cards should feel editorial and expressive.

⸻

Core UX Concept

Networking is centered around Creative Circles.

Instead of browsing profiles randomly, users discover people through shared creative context.

Creative Circles are based on:

• creative disciplines
• festival talks
• workshops
• shared interests

Examples:

Animation Circle
Game Design Circle
Motion Design Circle
Illustration Circle
Concept Art Circle
Visual Storytelling Circle

Users join circles to discover other creatives attending the same festival.

The goal is to facilitate real-world connections during the event.

Avoid interactions resembling:

• swipe gestures
• profile liking
• social media feeds

⸻

Screens to Generate

Generate a full mobile interface including the following screens.

⸻

Home Screen

A visually striking home screen focused on the current festival.

Top section:

Large Festival Hero Card displaying:

• festival name
• location
• event date

The hero should feel immersive and cinematic.

Below the hero:

Suggested Creative Circles.

Each card displays:

• circle title
• number of creatives inside
• skill tags

Cards should feature:

• neon gradient glow
• layered card depth
• subtle shadow diffusion

The layout should feel dynamic and editorial, not grid-based.

⸻

Circles Discovery Screen

A discovery page for Creative Circles.

The layout should resemble a creative directory or catalog.

Each circle card includes:

• circle title
• creative discipline
• participant count

Typography should dominate the visual hierarchy.

Minimal imagery.

Cards should feel curated and premium.

⸻

Circle Detail Screen

A hub where creatives within a circle discover each other.

Display a grid of attendees.

Each attendee card includes:

• name
• discipline
• skill tags

Example skill tags:

Blender
Concept Art
3D Animation
Game Design
Storyboarding
Unreal Engine

Skill tags should visually stand out more than profile photos.

This emphasizes networking through creative expertise.

⸻

Profile Screen

Design a creative professional profile page.

Sections include:

Name
Creative discipline
Skill tags
Portfolio links

Portfolio integrations may include:

Artstation
Behance
Personal Website

The layout should feel like a creative portfolio profile, not a social media account.

Prominently include a Connect button.

⸻

Connect Interaction

When pressing Connect, show a refined interaction overlay.

Options:

Save Contact
Open Portfolio
Exchange LinkedIn

The overlay should feel minimal, elegant, and quick.

⸻

Messages Screen

A minimal professional messaging interface.

Messaging should be secondary to real-world interaction.

Design should feel:

• quiet
• minimal
• distraction-free

Avoid typical social media chat UI patterns.

⸻

Motion & Micro-Interactions

Include subtle motion design concepts:

• card expansion transitions
• soft glow hover states
• smooth screen transitions
• gradient movement

Motion should feel:

• cinematic
• elegant
• refined

Avoid exaggerated animations.

⸻

Final Design Goal

Create a premium networking experience for creatives attending Playgrounds festivals.

The interface must feel:

• artistic
• editorial
• cinematic
• highly curated

It should look like a product built specifically for designers, filmmakers, animators, and digital artists.

The UI should feel like a natural extension of the Playgrounds brand and creative culture.
